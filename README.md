# PTA Open Hour — Portfolio Demo

An independent, browser-only demo of the original parent-teacher appointment system.

## Demo paths

- Family booking site: index.html
- Administrative console: admin.html

No login or email verification is required. The demo includes a fictional family profile, teachers, available times, existing appointments, booking confirmation, teacher management, and administrator controls. Email and Google Apps Script actions are simulated. Data is stored only in localStorage.
